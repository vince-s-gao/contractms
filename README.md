# 合同管理系统

## 项目简介

合同管理系统是一个企业级的合同全生命周期管理平台，支持合同的创建、审批、执行、归档等完整业务流程。

## 技术架构

### 前端技术栈
- **框架**: Vue 3.x + Element Plus
- **状态管理**: Pinia
- **路由管理**: Vue Router
- **构建工具**: Vite
- **HTTP客户端**: Axios
- **表单验证**: VeeValidate

### 后端技术栈
- **框架**: Spring Boot 3.x
- **ORM框架**: MyBatis-Plus
- **安全框架**: Spring Security
- **缓存**: Redis
- **消息队列**: RabbitMQ
- **API文档**: Swagger

### 数据库
- **主数据库**: MySQL 8.0+
- **缓存数据库**: Redis
- **搜索引擎**: Elasticsearch

## 快速开始

### 环境要求
- Node.js 18+
- Java 17+
- MySQL 8.0+
- Redis 7.0+

### 开发环境搭建

1. **前端开发环境**
```bash
cd frontend
npm install
npm run dev
```

2. **后端开发环境**
```bash
cd backend
./mvnw spring-boot:run
```

### 代码规范

### 分支管理
- main: 主分支，生产环境代码
- develop: 开发分支，集成测试环境

- hotfix/*: 热修复分支
